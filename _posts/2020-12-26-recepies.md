#### sql using regex
 Add a check constraint to the `id` column to enforce alphanumeric strings of exactly 5 characters long

```sql
ALTER TABLE short_urls ADD CONSTRAINT id CHECK (id ~ '^[a-zA-Z0-9]{5}$');
```
